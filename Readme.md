### Installing (for linux)

```
python3 -m venv venv
```
```
source venv/bin/activate
```
```
pip install --upgrade pip
```
```
pip install -r requirements.txt
```
```
flask db init
```
```   
flask db migrate -m "entries table"
or
flask db migrate 
```
```
flask db upgrade
```
```
flask run
```
